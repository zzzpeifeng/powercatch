## 移除方案 B（流量过滤器）

**原因**：系统代理开启后，macOS 系统代理（127.0.0.1）作为中间人转发所有流量，导致 proxy 看到的 clientIp 全部是 127.0.0.1，无法区分本机流量和移动设备流量。

**移除内容**：
- `RequestList.vue` — 移除 trafficFilter prop、isLocalTraffic 函数、流量类型 Tab UI、"本机"标签
- `MainView.vue` — 移除 trafficFilter 状态、handleApplyFilter 函数、localStorage 持久化
- `SystemProxyBanner.vue` — 移除"只看手机"按钮和 apply-filter emit

**验证**：vue-tsc 0 错误 + npm run build 成功
