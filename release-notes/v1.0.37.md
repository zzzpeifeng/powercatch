# v1.0.37 - DNS 覆盖

## 🎯 新功能

### #16 DNS 覆盖
在代理层将域名指向自定义 IP，不需要修改系统的 hosts 文件。

**功能特性**：
- ✅ 添加/删除/启用/禁用 DNS 覆盖规则
- ✅ 规则格式：域名 → IP 地址
- ✅ 支持通配符域名（如 *.example.com）
- ✅ 规则持久化存储
- ✅ 代理层自动应用规则
- ✅ 保持 Host 头为原始域名（确保虚拟主机正常工作）

**入口**：工具下拉菜单 → DNS 覆盖

**典型使用场景**：
1. 线上域名指向测试环境 IP
2. 本地开发时绕过 DNS 解析
3. 多环境切换（dev/staging/prod）
