# PowerCatch v1.0.23 Release Notes

**发布日期**: 2026-06-20

---

## ✨ 新功能与改进

### 1. #1 高级过滤系统
6 维度过滤，快速定位目标请求：

- **HTTP 方法过滤**：GET/POST/PUT/DELETE/PATCH/HEAD/OPTIONS
- **状态码分组**：2xx/3xx/4xx/5xx/pending
- **Content-Type 分组**：json/html/image/javascript/css/other
- **响应时间范围**：fast(<100ms)/normal/slow/very_slow/pending
- **请求体大小范围**：empty/tiny/small/medium/large
- **设备 IP 过滤**：按客户端 IP 过滤

### 2. UI 修复
- 修复主题切换后部分组件颜色不一致
- 修复退出时系统代理未恢复的问题

## 📋 安装说明

1. 下载 `PowerCatch-1.0.23-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
