# v1.0.40 - GraphQL 感知

## 🎯 新功能

### #23 GraphQL 感知
识别 GraphQL 请求，并按 operation name 分组显示。

**功能特性**：
- ✅ 自动识别 GraphQL 请求（POST + Content-Type + query 字段三重校验）
- ✅ 解析 operation name（优先 JSON operationName，回退从 query 解析）
- ✅ 解析 operation type（query/mutation/subscription）
- ✅ 请求列表显示 GraphQL operation name 标签（按类型着色：蓝/橙/紫）
- ✅ 过滤面板新增 GraphQL 操作过滤（5 种维度）

**入口**：自动识别，过滤面板 → GraphQL 操作

**典型使用场景**：
1. 识别 GraphQL 请求，区分 query、mutation、subscription
2. 按 operation name 分组显示（如 getAllUsers、createOrder）
3. 快速过滤特定类型的 GraphQL 操作
