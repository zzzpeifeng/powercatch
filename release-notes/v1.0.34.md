# v1.0.34 - Auto Responder（自动响应器）

## 🎯 新功能

### #10 Auto Responder（自动响应器）
不请求服务器，直接返回预设的响应内容。支持模拟登录成功/失败、网络错误、慢响应等场景。

**功能特性**：
- ✅ 规则管理：添加、编辑、删除、启用/禁用、全部启用/禁用、清空
- ✅ 响应配置：状态码、Headers（key-value 编辑器）、Body（多行文本）、延迟（毫秒）
- ✅ URL 匹配：复用 breakpoint-matcher 的通配符匹配逻辑
- ✅ 优先级：断点 > Map Local > Auto Responder > Map Remote
- ✅ 数据持久化：SQLite 数据库 + 设置存储双重持久化
- ✅ UI 设计：参考 MapLocalRules.vue，支持亮/暗模式

**入口**：工具下拉菜单 → Auto Responder

**典型使用场景**：
1. 模拟登录成功/失败响应
2. 模拟网络错误（500/502/503）
3. 模拟慢响应（延迟 3 秒）
4. 前端独立开发，不依赖后端
