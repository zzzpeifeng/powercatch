# PowerCatch v1.0.8 Release Notes

**发布日期**: 2026-06-15

---

## ✨ 新功能与改进

### 1. 关闭窗口行为优化（方案C）
- 点击窗口红色关闭按钮时，应用**隐藏到 Dock** 而非完全退出
- 点击 Dock 图标可快速恢复窗口，无需重新启动
- 真正退出需使用 `Cmd+Q` 或右键 Dock → 退出

### 2. 证书安装指引文案优化
- 统一证书文件名为 `ca.pem`（实际文件名）
- 更新证书名称为 `NodeMITMProxyCA`（证书内部 CN）
- 下载文件名改为 `NodeMITMProxyCA.pem`（更直观）

## 📋 安装说明

1. 下载 `PowerCatch-1.0.8-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
