# v1.0.29 - Map Remote 远程映射

## 新功能

### #9 Map Remote（远程映射）

将匹配指定 URL 的请求转发到另一个域名/地址，实现环境切换（如线上→测试环境）。

**核心能力**：
- **URL 模式匹配**：支持 `*` 通配符（如 `*api.shopline.com*`）
- **HTTP 方法过滤**：可选按 GET/POST/PUT/DELETE 等方法过滤
- **目标地址配置**：协议（http/https）+ 主机 + 端口 + 路径替换
- **启用/禁用切换**：每条规则可独立启用或禁用
- **规则管理 UI**：完整的增删改查界面

**与 Map Local 的区别**：
- Map Local：返回本地文件，不转发到服务器
- Map Remote：修改目标地址后继续转发到服务器

**入口**：工具栏 → 工具 下拉菜单 → Map Remote

## 修复

- 修复 Map Local 规则弹窗"浏览"按钮无响应的问题
- 修复 Map Remote 规则弹窗目标地址输入框布局问题

## 技术细节

- 拦截机制在 Map Local 检查之后、断点检查之前执行
- 支持 TLS 协议切换（HTTP↔HTTPS）
- 规则持久化到 SQLite（`map_remote_rules` 字段）
- 与 Map Local 共用工具栏下拉菜单，蓝色徽章显示规则数量
