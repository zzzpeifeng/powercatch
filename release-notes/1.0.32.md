# PowerCatch v1.0.32

## 🚀 AI 分析 v2.0 - 双阶段 Pipeline + JSON 容错解析

### ✨ 新功能

- **双阶段 AI Pipeline**：Code Explorer（代码探索）→ Test Generator（用例生成），场景数量由代码复杂度动态决定
- **无工具调用上限**：Phase 1 不再限制工具调用次数，10 分钟超时保护，让 AI 充分探索代码
- **智能 JSON 容错解析**：
  - 支持 ```` ```json ```` 代码块提取
  - 括号配对追踪，截断自动补全
  - 转义控制字符（真实换行 → `\n`）
  - 修复 trailing commas、未加引号的 key
  - 正则提取部分字段作为降级方案
- **deepseek 推理模型支持**：读取 `reasoning_content` 字段，`testConnection` 兼容推理模型
- **嵌入式 Prompt 管理**：Prompt 文件嵌入 TypeScript，避免 Vite 打包路径问题

### 🐛 修复

- 修复 Phase 2 AI 输出未转义控制字符导致 JSON 解析失败的问题
- 修复 `testConnection` 对推理模型（如 deepseek-v4-pro）误判为"无响应"的问题
- 修复流式响应漏掉推理模型 `reasoning_content` 内容的问题
- 移除 `pushLog` 终端打印，减少日志干扰

### 📦 技术改进

- 新增 `prompt-loader.ts`：嵌入式 Prompt 管理，支持变量替换
- 新增 `types.ts`：`CodeExplorationResult` 内部类型定义
- `parseExplorationResult` 完全重写：三层容错（直接解析 → 修复 → 正则提取）
- `parseScenariosResult` 同样加固：复用智能 JSON 提取 + 修复逻辑
- Phase 2 Prompt 增强：明确 JSON 转义规范，附错误/正确示例
