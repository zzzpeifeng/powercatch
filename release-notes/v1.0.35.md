# v1.0.35 - 请求重放 & 编辑重发

## 🎯 新功能

### #8 请求重放 & 编辑重发
支持直接重发请求和编辑请求参数后重发，方便调试和测试 API。

**功能特性**：
- ✅ 右键菜单 → "重发请求"：一键重发选中的请求
- ✅ 右键菜单 → "编辑后重发"：弹窗编辑 URL/Method/Headers/Body 后重发
- ✅ 重发结果展示：状态码、耗时、响应头、响应体预览
- ✅ 支持 HTTP/HTTPS
- ✅ 30 秒超时保护

**入口**：请求列表右键菜单 → 重发请求 / 编辑后重发

**典型使用场景**：
1. 调试 401/403 错误：修改 token 后重发
2. 测试参数边界：修改请求体后重发
3. 复现问题：快速重发请求
4. API 测试：修改参数后重发
