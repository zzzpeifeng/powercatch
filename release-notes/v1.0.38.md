# v1.0.38 - 多格式导出

## 🎯 新功能

### #21 多格式导出
将选中的请求导出为多种格式，方便在其他工具中使用。

**功能特性**：
- ✅ cURL 命令 → 复制到剪贴板
- ✅ Postman Collection v2.1 → 下载 .json 文件
- ✅ JMeter 脚本 → 下载 .jmx 文件
- ✅ fetch (JavaScript) → 复制到剪贴板
- ✅ Python requests → 复制到剪贴板

**入口**：请求右键菜单 → 导出为 → 选择格式

**典型使用场景**：
1. 将抓包到的请求导出为 Postman Collection，方便团队共享
2. 将请求导出为 JMeter 脚本，进行性能测试
3. 将请求导出为代码，方便开发调试
