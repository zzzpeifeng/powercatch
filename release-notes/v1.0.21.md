# PowerCatch v1.0.21 Release Notes

**发布日期**: 2026-06-18

---

## ✨ 新功能与改进

### 1. Structure 树状视图
新增树状视图模式，按域名分组显示请求：

- **域名分组**：自动按 host 分组，显示请求数量
- **展开/折叠**：支持展开/折叠域名节点
- **虚拟滚动**：大量请求时保持流畅
- **搜索过滤**：支持 URL 搜索和过滤

### 2. 协议前缀修复
- 修复 URL 中协议前缀（http:// / https://）显示问题
- 修复 HTTPS 请求 URL 解析错误

## 📋 安装说明

1. 下载 `PowerCatch-1.0.21-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
