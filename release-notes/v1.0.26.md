# PowerCatch v1.0.26 Release Notes

**发布日期**: 2026-06-22

---

## ✨ 新功能与改进

### 1. #6 断点 & 请求/响应修改
强大的请求拦截和修改功能，支持在请求/响应阶段暂停并编辑数据。

- **断点规则管理**：支持 URL 通配符匹配 + HTTP 方法过滤
- **请求阶段拦截**：暂停请求，编辑请求头/请求体后放行
- **响应阶段拦截**：暂停响应，编辑响应头/响应体后放行
- **右键菜单**：右键点击请求可快速添加断点规则
- **全局模态弹窗**：断点编辑使用全局模态框，支持排队处理
- **cURL 命令生成**：一键复制请求为 cURL 命令

### 2. 工具栏 UI 优化
- 三段式布局（左/中/右）
- 统一 light:dark 双主题配色
- 断点按钮带红色徽章显示规则数

### 3. 断点规则面板优化
- 模态弹窗形式（遮罩层 + 点击遮罩关闭）
- 配色统一使用 CSS 变量
- 窗口尺寸扩大到 640px

## 📋 安装说明

1. 下载 `PowerCatch-1.0.26-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
