# PowerCatch v1.0.7 Release Notes

## 🎯 版本概要

**发布日期**: 2026-06-15  
**版本号**: v1.0.7  
**平台支持**: macOS arm64 (Apple Silicon)

## ✨ 主要变更

### 修复：DMG "已损坏" 问题
- **问题**: 将 DMG 发给其他人安装时，macOS Gatekeeper 会提示"已损坏"，无法打开
- **原因**: 之前的版本没有进行代码签名，macOS 安全机制拦截了未签名的 App
- **修复**: 配置 ad-hoc 签名（`identity: "-"`），打包时对 App 进行 ad-hoc 签名
- **效果**: 发给别人后，对方可以直接安装打开，不会再提示"已损坏"

### 技术细节
- 修改 `electron-builder.json` 的 `mac.identity` 配置
- Ad-hoc 签名不需要 Apple Developer 证书，适合内部分发
- 如果对方 Mac 安全设置较严格，可能仍需要在"系统设置 > 隐私与安全性"中允许

## 🚀 安装步骤

1. 下载 `PowerCatch-1.0.7-arm64.dmg`
2. 双击打开 DMG 文件
3. 将 PowerCatch 拖入"应用程序"文件夹
4. 首次打开时，如果提示"无法验证开发者"：
   - 打开"系统设置 > 隐私与安全性"
   - 滚动到底部，点击"仍要打开"
   - 再次点击"打开"确认
