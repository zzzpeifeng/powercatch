# PowerCatch v1.0.28 Release Notes

**发布日期**: 2026-06-22

---

## ✨ 新功能与改进

### 1. #7 Map Local（本地映射）
将指定 URL 的响应替换为本地文件内容，无需请求真实服务器。常用于前端开发中模拟 API 响应、调试时替换 JS/CSS 资源。

- **规则管理面板**：模态弹窗形式，支持新增/编辑/删除规则
- **URL 匹配引擎**：支持通配符模式（如 `*api.example.com/users*`）
- **优先级**：Map Local 优先于断点检查，命中后直接返回本地文件
- **文件支持**：支持文本文件（JSON/JS/CSS/HTML）和二进制文件（图片/字体）
- **持久化**：规则自动保存，重启后自动加载

### 2. 工具栏下拉菜单整合
优化工具栏布局，将功能按钮整合到下拉菜单中：

- **工具菜单**：断点 + Map Local（带徽章显示规则数）
- **AI 处理菜单**：AI 对比功能
- **通用组件**：新增 DropdownMenu 组件，支持过渡动画和点击外部关闭

## 📋 安装说明

1. 下载 `PowerCatch-1.0.28-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
