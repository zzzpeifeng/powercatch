# PowerCatch v1.0.11 Release Notes

**发布日期**: 2026-06-16

---

## ✨ 新功能与改进

### 1. 系统代理集成
- 支持一键设置/清除系统代理
- 支持检测系统代理状态
- 退出时自动恢复系统代理

### 2. 主题切换
- 支持 light/dark/system 三种主题
- 主题设置持久化

## 📋 安装说明

1. 下载 `PowerCatch-1.0.11-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
