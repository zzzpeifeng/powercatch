# PowerCatch v1.0.19 Release Notes

**发布日期**: 2026-06-18

---

## ✨ 新功能与改进

### 1. SSL 错误处理优化
- 新增 SSL 错误分类和诊断报告
- 支持证书固定域名配置
- 优化 HTTPS MITM 握手失败处理

### 2. WiFi 自动配置
- 支持生成 WiFi 代理配置
- 支持 QR 码扫描配置

## 📋 安装说明

1. 下载 `PowerCatch-1.0.19-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
