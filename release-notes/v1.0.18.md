# PowerCatch v1.0.18 Release Notes

**发布日期**: 2026-06-17

---

## ✨ 新功能与改进

### 1. 设备别名功能
- 支持为客户端 IP 设置别名
- 别名显示在请求列表中
- 别名持久化存储

### 2. 对比记录历史
- 保存 AI 对比结果到数据库
- 支持查看对比历史记录

## 📋 安装说明

1. 下载 `PowerCatch-1.0.18-arm64.dmg`
2. 打开 DMG，将 PowerCatch 拖入「应用程序」文件夹
3. 首次打开若提示"无法验证开发者"：
   - 前往「系统设置 → 隐私与安全性」
   - 滚动到底部，点击「仍要打开」
