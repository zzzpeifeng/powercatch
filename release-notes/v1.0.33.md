# v1.0.33 - 请求 Diff 视图

## 新功能

### #12 请求 Diff 视图

选择两个 HTTP 请求进行**结构化对比分析**，快速识别差异。

**核心能力**：
- **Headers 对比**：并排展示请求头/响应头差异，忽略大小写
- **Body 对比**：JSON 结构化 diff（递归对比 + delta 格式）+ 文本逐行 diff（LCS 算法）
- **三色标记**：绿色(新增) / 红色(删除) / 黄色(修改)
- **概览 Tab**：统计卡片 + 差异列表，30 秒定位问题
- **导出功能**：HTML 报告 / Markdown 表格 / JSON 结构化数据
- **状态持久化**：页面刷新后保留对比结果

**入口**：工具栏 → 工具 下拉菜单 → Diff 视图

**使用方式**：
1. 勾选 2 个请求（复选框）
2. 点击工具 → Diff 视图
3. 查看对比结果

## 技术细节

- Diff 算法纯手写实现，不依赖外部库（jsondiffpatch/diff）
- JSON diff 使用递归对比 + delta 格式
- 文本 diff 使用 LCS 算法（大数据降级为简化版）
- 状态管理：Pinia store + sessionStorage 持久化（key: `powercatch:diff-state`）
- 页面缓存：Vue `<keep-alive>` 只缓存 DiffView
- 新增文件：`diff-engine.ts` / `diff-store.ts` / `diff-export.ts` / `DiffView.vue` / `DiffOverview.vue` / `DiffHeaders.vue` / `DiffBody.vue`
