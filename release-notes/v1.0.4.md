# PowerCatch v1.0.4 Release Notes

> 发布日期：2026-06-15

---

## 🆕 新功能

- **打包版本自动递增**：每次执行 `npm run electron:build` 时，Patch 版本号会自动 +1（如 `1.0.3` → `1.0.4`），无需手动修改 `package.json`

---

## 🐛 Bug 修复

- **修复 CA 证书路径错误**：解决 `getSslCaDir()` 返回路径错误（`certs` → `ssl-ca`），以及 `mitm-server.ts` 中硬编码路径导致证书目录不正确的问题
- **修复 KEY_VALUES_MISMATCH 错误**：修复 CA 证书与私钥不匹配导致 HTTPS 抓包失败的问题
- **修复设置无法持久化**：修复「基础设置」页面点击保存后，重启软件配置丢失的问题（`handleSave()` 未将 `localSettings` 同步到 `settingsStore`）
- **修复 theme 字段未持久化**：新增 `theme` 字段的数据库读写逻辑，主题设置现在可以正确保存和读取
- **修复证书安装向导步骤判断**：解决已存在 CA 证书时，向导仍显示「步骤 1：生成证书」的问题，现在会优先从磁盘检查真实状态，直接跳到安装步骤

---

## 🔧 优化改进

- **软件标题统一**：窗口标题、自定义标题栏、证书下载页、导出报告落款统一改为 **PowerCatch**
- **项目名统一**：内部 `package.json` 的 `name` 字段统一为 `powercatch`，用户数据目录路径与之保持一致

---

- `PowerCatch-1.0.4-arm64.zip`
